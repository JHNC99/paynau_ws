## Instalar proyecto

```bash
$ npm install
```

## Compilar y ejecutar el proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Configurar .env
Clonar el archivo .env.template y renombrarlo a .env

```bash
$ cp .env.template .env 
```
